# React Native Hybrid App

## 실행 방법

### 1. 환경 설정
```bash
# 의존성 설치
npm install
```

### 2. 실행하기
```bash
# Expo 개발 서버 실행
npx expo start
```

### 3. 앱 실행
- **모바일 기기에서 테스트**
  - iOS: 카메라 앱으로 QR 코드 스캔
  - Android: Expo Go 앱으로 QR 코드 스캔

- **시뮬레이터에서 테스트**
  - iOS: 터미널에서 `i` 입력
  - Android: 터미널에서 `a` 입력

앱이 실행되면 PWA가 WebView 내에서 자동으로 로드됩니다.
